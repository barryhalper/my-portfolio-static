Municipal Yearbook : Public Spend
Visit website Play video View gallery
A B2B application that aggregates spending data published by UK Councils over the last 5 years, presenting it to subscribers in a suite of custom-built web reporting tools. The project consists of a subscriber-only web application and a CMS/CRM that allows the data team to upload data from XML and CSV artefacts and manage user fulfilment. Built on the Microsoft stack of SQL Server 2016, ASP.NET framework, MVC and Entity Framework, and utilising test-driven development, the application includes various data warehousing, indexing and caching techniques as well as JQuery and a Rest API to optimise the performance of a data set of over 48,000,000 rows. This ensures the complex algorithms that aggregate data can render efficiently a range of customer-defined, reports, d3 charts, and datasets. As well as building all the database architecture and applications I also created the UX using good practice web design, based on industry standards of HTML5, Bootstrap, Javascript and contributed to the marketing site created by the front end team.

Hemming Publishing CMS
Play video View gallery
With over 20 years of publishing experience as both writer and developer, I was in a unique position to build a complete (CMS) content management system deployed across the Hemming Group content portfolio. The latest version utilises Microsoft .Net MVC with a domain-driven architecture, Entity Framework, SQL Server and Javascript ES6. The application can be easily set up for future projects enabling (RAD) rapid application development for the team while giving the content provider a tool whose UX has been designed specifically for their needs. Out of the box, the application has a myriad of tools that allows data to be imported from various sources, such as print media, Web APIs, CSV, XML and various third-party providers of subscription fulfilment (CRM). It also manages and reports on orders, customer input, advertising, SEO and analytics. Best standards are utilised for data security and integrity.

Municipal Yearbook
Visit website Play video View gallery
This directory is a market leader in online repositories providing the most current information on all UK local government officers and elected officials. For this project, I created and managed a subscription-based web application, an editorial CRM and a client-facing RESTFUL API. The SQL database and .NET web application features payment based access and entitlement (using Stripe), order progression and subscription fulfilment as well as complex data analysis and reporting. The web application provides an intuitive user interface to help users access the information quickly and efficiently and has a customisable dashboard so subscribers can target key information to help make smart business decisions. To this end, I employed a team of designers and third party testers to deliver this project on time and on budget. I also worked closely with the editorial departments to create a CRM to manage the regular cycle of data entry and campaign management. I also created and ran the process that enabled editorial to export the data from SQL to Indesign for physical publication.

Localgov

The UK's largest dedicated local government news website. I led a team of contractors to deliver this project using Coldfusion, Javascript, Ajax and MS SQL server. The project includes a custom-built CMS Full-Text searches, a REST API and integrations tools with third-party providers. The design template for the site has been reused for many other titles in the Hemming portfolio.
